# Security Policy

## Supported versions

Security fixes are provided for the latest release line of ChanchiNutri.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately through GitHub private vulnerability reporting (preferred).

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

## Response expectations

- Initial acknowledgment target: within 72 hours
- Triage and severity assessment: as soon as possible
- Fix timeline depends on severity and complexity

We appreciate responsible disclosure.
