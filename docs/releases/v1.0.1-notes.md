## Resumen para usuarios
Esta actualizacion mejora la calidad de formulacion en la app: ahora ves alertas mas claras sobre limites de inclusion por ingrediente y se ampliaron los controles nutricionales para ayudarte a ajustar dietas con mayor precision.

## Cambios funcionales visibles
- Se agregaron nuevas metricas nutricionales visibles en calculos y validaciones: triptofano, valina e isoleucina.
- Ahora la pantalla de creacion de dieta muestra el rango recomendado por ingrediente y avisa cuando un porcentaje queda por debajo o por encima del limite.
- En la pantalla de resultados, las recomendaciones se separan por tipo (limites de inclusion y balance nutricional) para facilitar la interpretacion.

## Mejoras tecnicas internas (breve)
- Se reforzaron pruebas automaticas y controles de calidad para publicar versiones con menor riesgo de errores.
- Se incorporaron automatizaciones para validar artefactos de release antes de publicar.

## Compatibilidad e instalacion
- Esta version mantiene compatibilidad con el flujo de instalacion Android ya conocido en EvaPigApp.
- Si venis de `v1.0.0`, solo tenes que instalar/actualizar la APK de `v1.0.1`.
